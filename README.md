# (Major Project) Intelligent-Decision-Support-System
## Overview
This project is an End-to-End Intelligent Decision Support System (IDSS) developed for Walmart Retail Analytics. It integrates Machine Learning, SQL Server, Power BI, and Python to analyze sales, forecast demand, optimize inventory, and generate business decision recommendations.

## Problem Statement
Retail businesses often struggle with demand forecasting, inventory optimization, and decision-making due to large volumes of sales data. This project provides a data driven decision support system that helps improve forecasting accuracy and inventory management.

## Objectives
- Analyze Walmart sales data
- Forecast future sales using Machine Learning
- Generate inventory recommendations
- Build interactive Power BI dashboards
- Support business decision making using data analytics

## Tech Stack
- Python
- Machine Learning
- SQL Server
- Power BI

## Machine Learning Models
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training & Evaluation
6. Sales Forecasting
7. Inventory Decision Support
8. SQL Database Integration
9. Power BI Dashboard

## Dashboard Features
- Executive Overview
- Sales & Store Analysis
- Demand Forecasting
- Inventory Risk Analysis

## Results
- Accurate sales forecasting using Machine Learning
- Interactive Power BI dashboards
- Inventory optimization recommendations
- Business decision support through data-driven insights
